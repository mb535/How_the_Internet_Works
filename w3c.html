W3C HTML and CSS

HTML and CSS are considered the frontend development tools and languages of a website dealing the with how the website looks and navigates.

W3C (World Wide Web Consortium) is the main organization body that manages the World Wide Web. It was founded by Tim Berners-Lee in 1994. W3C's primary activity is to develop protocols and guidelines that ensure long-term growth for the Web. W3C develops the standards that define key parts of what makes the World Wide Web work.

HTML (Hyper Text Markup Language) is a markup language you use to tell a web browser how to make a page look. It is one of the languages used to design the aesthetics of the page. HTML pages usually consists of a bunch of different tags which customizes looks and layout of the page.

CSS (Cascading Style Sheets) is a language which is often used in conjunction with HTML to further improve the looks and layout of HTML pages. CSS is designed to make customization more modular, flexible, and accessible by making the content and presentation separate.
