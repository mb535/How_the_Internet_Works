DNS (Domain Name System)

DNS servers are usually located in data centers

DNS (Domain Name System) is a server that handles domain name resolution. In other words in translates the domain name with it corresponding assigned IP address. For example, a DNS server associate names such as www.example.com with its corresponding ip address such as 174.129.14.201. The computer uses the DNS server to look domain names and get the associated IP address which is used to connect the computer to the destination on the internet. There are numerous DNS servers around the world which are connected in a distributed hierarchy and are divided into ones, splitting up responsibility for the major domains such as .org,.com,.net,etc. One of the most popular DNS server right now is Google which has an ip address of 8.8.8.8.
